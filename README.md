# shell-scripts
***Description***

1. txt2pdf.sh
	This shell script converts any postscript file to pdf.
2. altf4.sh
	This shell script does some operations related to power managment like power-off, log-out, sleep,....etc.
3. petable.sh
	This shell script gives some basic info. about elemnts.

	Note: Before running petable.sh you need to run install.sh to install some of the packages.
